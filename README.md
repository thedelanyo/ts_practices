# 📊 Stats Tool: Rust to Bun JS Migration

This project contains small utility programs built while learning and mastering JS/TS concepts using (Bun)[https://bun.com/].


## 🛠 Tech Stack
 * **Runtime:** Bun (v1.0+)
 * **Language:** TypeScript

## 📂 Installation & Usage

### Prerequisites
You must have Bun installed on your machine.

### Running the Program
 1. **Clone the repo:**
   ```bash
   git clone https://github.com/your-username/stats-tool.git
   cd stats-tool

   ```
 2. **Install dependencies (if any):**
   ```bash
   bun install

   ```
 3. **Run the script:**
   ```bash
   bun run index.ts

   ```
